JavaSE基础 day12

异常

概述

异常：在运行时发生的不正常情况。
在java中用类的形式对不正常的情况进行描述和封装对象。描述不正常的情况的类，就成为异常类。
以前正常流程代码和问题处理代码相结合，现在将正常流程代码和问题处理代码分离，提高阅读性。
其实异常类就是java通过面向对象的思想将问题封装成了对象。用异常类对其进行描述。
不同的问题用不同的类进行具体的描述。比如角标越界，空指针等。

JavaSE基础 day10

内部类

概述

  • 将一个类定义在另一个类里面，对里面那个类就称为内部类（内之类，嵌套类）。
  • 访问特点：
    • 内部类可以直接访问外部类中的成员，包括私有成员。
    • 而外部类要访问内部类中的成员必须要建立内部类的对象。

作用：
一般用于用于类的设计。
分析事物时，发现该事物A描述中还有事物B，而且这个事物B还在访问被描述事物A的内容，这时候就可以将事物B定义成内部类来描述。

JavaSE基础 day04

面向对象

  • 面型过程，强调的是动作 c语言
  • 面向对象，强调的是实体 c# java c++
    面向对象的特点
  • 面向对象是一种常见的思想，符合人们的思考习惯。
  • 面向对象的出现，将复杂问题简单化。
  • 让曾经在过程中的执行者变成了对象中的指挥者。
    举例：面试官为了提高效率，需要更多具备专业能力的人帮助您做事，这就是您所需要的对象，调用对象的开发功能完成项目。

类与对象的关系

  • 用JAVA语言对现实生活中的实物进行描述，通过类的形式实现。
  • 只关注两个方面，属性（成员变量），行为（成员函数）。
  • 明确事物的属性行为后定义到类中即可。
  • 对象：实实在在存在的个体。
