JavaSE基础 day04

面向对象

  • 面型过程，强调的是动作 c语言
  • 面向对象，强调的是实体 c# java c++
    面向对象的特点
  • 面向对象是一种常见的思想，符合人们的思考习惯。
  • 面向对象的出现，将复杂问题简单化。
  • 让曾经在过程中的执行者变成了对象中的指挥者。
    举例：面试官为了提高效率，需要更多具备专业能力的人帮助您做事，这就是您所需要的对象，调用对象的开发功能完成项目。

类与对象的关系

  • 用JAVA语言对现实生活中的实物进行描述，通过类的形式实现。
  • 只关注两个方面，属性（成员变量），行为（成员函数）。
  • 明确事物的属性行为后定义到类中即可。
  • 对象：实实在在存在的个体。
