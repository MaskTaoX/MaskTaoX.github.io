JAVA

函数-格式

定义函数的格式
修饰符 返回值类型 函数名（参数类型 形参1.参数类型 形参2….）
{ 

执行语句;
return 返回值;

}
函数名称要具有可读性
第一个单词小写剩下的全部大写

函数-细节

  • 函数只能调用函数，不可以在函数内部定义函数
  • 定义函数时，函数的结果要返回给调用者

函数-两个明确

思路：既然是功能定义，就是可以用户函数来体现。
我们通过两个明确来完成函数定义

  • 明确1：这个功能结果是什么
  • 明确2：这个功能是否需要未知内容来参与运算

函数重载

在同一各类中，允许存在一个以上的同名函数，他们只需要参数个数或者参数类型不同即可

数组内存划分

![image]（）

  • 寄存器
  • 本地方法区
  • 方法区
  • 栈内存：存储的都是局部变量（定义在方法中的变量）而且变量所属作用域一旦结束，该变量就自动释放。
  • 堆内存：
    • 存储的是数组和对象，凡是new建立的都在堆中；
    • 每一个实体都有内存地址，有首地址值；
    • 实体中的变量都有默认初始化值实体不在被使用，会在不确定的时间内被垃圾回收器回收；

      int[] x=new int[3]; 对其内存的描述
      1.主函数先进栈，运行到数组定义
      2.在堆中给实体分配内存地址,默认初始化值
      3.堆中地址由栈中x操作

注意

此时x[0]=34,
因为y把地址赋给x，x此时也指向y所指向的内存
