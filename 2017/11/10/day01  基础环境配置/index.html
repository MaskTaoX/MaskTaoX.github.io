JavaSE基础 day01

JAVA
  • 学习

1.1 人机交互

三种技术架构

  • JavaEE 企业
  • JavaSE 标准
  • JavaME 小型

1.2 JAVA 语言的跨平台性

通过虚拟机 JVM 根据不同的平台安装虚拟机版本

1.3 JAVA语言环境搭建

  • jre （java runtime environment）java运行环境 （包括JVM，类库）
  • jdk （java development kit）java工具包

安装了jdk就不用安装jre了

java开发工具包包括1.编译工具（javac.exe）2.打包工具（jar.exe）

javaFX SDK 脚本语言 应用于web 图形界面制作 是一种前端语言

1.5 jdk命令行工具

命令存放在bin目录，即jdk的开发工具

1.6 命令行简介

dos命令行

  • dir：列出当前目录下的文件
  • rd：删除目录
  • md：创建目录
  • del：删除目录
  • cd：进入指定目录
  • help：显示所有命令的注解

1.7环境变量的配置

  • 在任何人目录下都可以执行命令 对系统变量path的操作
  • 找到jdk/bin的路径，将其复制到电脑属性–高级系统设置–环境变量–系统变量–找到path并添加jdk/bin的路径
  • 检查路径配置成功 win+r 任意路径下键入javac

1.8 环境配置技巧

  • 新建一个环境变量JAVA_HOME=d:\All Java In \jdk-9.0.1
  • path=%JAVA_HOME%\bin 。加%会被识别为环境变量

    1.10 java语言

  • 编辑：后缀txt保存为.java。（保存的是源代码，需要翻译成计算机语言）
  • 翻译：用javac进行编译 DOS命令“javac 123.java”—生成类名.class
  • 运行：用java运行 DOS命令“java 类名”

1.13classpath环境变量

  • set classpath=c:\myclass//虚拟机去此地寻找，找不到报错！
  • set classpath=c:\myclass;//虚拟机去此地找，找不到在当前目录再找一次
  • classpath就是规定虚拟机去哪执行程序
