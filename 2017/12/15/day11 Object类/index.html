JavaSE基础 day11

2017-12-15

Object类

object:所有类的根类。
Object是不断抽取而来的，具备着所有对象都具备的共性内容。

equals方法

两个对象是否相等。返回值是布尔类型。
比较的是地址

运行结果：
false
false
true

equals方法覆盖

不想比较地址值，要比较类中的某个变量值。
一般都会覆盖此方法，根据对象的特有内容，建立判断对象是否相同的依据。

hashCode方法

当equals方法被重写时，通常有必要重写hashCode方法，以维护hashCode方法的常规协定，该协定声明相等对象必须具有相等的哈希码。
在集合框架中会由重要的应用

getClass方法

返回此object的运行时类。
读到Person.class文件时堆中先加载Person.class字节码文件对象。

运行结果：
true
Person

toString方法

返回该对象的字符串表示。通常，toString方法会返回一个“以文本方式表示”此对象的字符串。建议所有子类都重写该方法。
